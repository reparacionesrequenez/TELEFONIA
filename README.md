# Sistema de Reparaciones en la Nube

Proyecto desarrollado en PHP y MySQL.

## Funciones
- Inicio de sesión
- Registro de equipos
- Edición de equipos
- Eliminación de equipos
- Gestión de usuarios

## Estructura
- `config/` conexión a base de datos
- `includes/` archivos reutilizables
- `public/` archivos públicos del sistema
- `database/` respaldo SQL

## Uso local
Abrir en:
`http://localhost/crud/public/`

## Tecnologías
- PHP
- MySQL
- XAMPP
- GitHub
